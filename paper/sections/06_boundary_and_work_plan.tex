\section{Boundary and work plan}

This paper begins as a conjectural native-generator bridge, not as a Gap A closure theorem.

It currently asserts only the reduced straightening result:

\[
    visible\ twist \quad\longrightarrow\quad hidden\ identity.
\]

It does not currently assert:
\begin{enumerate}
\item Gap A closure;
\item a complete G60-native generator;
\item derivation of the full role-labeled shared\_B edge universe;
\item derivation of the reduced 16-candidate universe from first principles;
\item physical interpretation.
\end{enumerate}

The next audit sequence should test the following questions.

\begin{enumerate}
\item Can the four C-readout states be generated from a single local answer cell?
\item Can the four anchor-readout states be generated from the same cell by a twist or wrap-carry motion?
\item Is the visible order \([B0,O0,O1,B1]\) forced by branch-anchor motion?
\item Is the fixed state \(B1\) a terminal branch state?
\item Is the three-cycle \((O0\ O1\ B0)\) a native wrap-carry phenomenon?
\end{enumerate}

A successful answer to these questions would not automatically close Gap A, but it would remove the main upstream dependency from the Project 21 reduced theorem.
