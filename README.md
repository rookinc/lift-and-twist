# Lift & Twist

Project 22 in the Thalean graph theory workspace.

This project is a companion to Project 21, `21-gap-a-answer-pair-generator`.

Project 21 proved a reduced-universe shell-rank selector inside the 16-candidate shared_B reduced universe. It did not close Gap A. The upstream problem remains to derive the reduced 16-candidate universe, or the full role-labeled shared_B edge universe, from native provenance.

This project studies the next conjectural mechanism:

Lift & Twist.

Working thesis:

The visible shared_B selector is not fundamentally a lookup table or scalar threshold. It is the visible shadow of a lifted identity rule.

In lifted shell/rank coordinates, the selector is:

    same shell
    same rank

In visible register coordinates, that identity appears as the permutation:

    [1, 2, 0, 3] = (0 1 2)(3)

The strong conjecture is that the reduced 16-candidate universe itself is generated by applying Lift & Twist to one local answer cell, rather than by choosing observed C-cycle support and observed anchor-cycle support after the fact.

Boundary:

This project does not claim Gap A closure at the outset. It begins as a coordinate-straightening and native-generator conjecture.
